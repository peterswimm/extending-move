# Setup Guides for Extending Move

Welcome! This folder contains user-friendly setup guides for installing Extending Move on your Ableton Move device. Choose the guide that matches your computer's operating system.

## Before You Start

⚠️ **Important Safety Notice**: This process involves accessing your Move device in ways it wasn't originally designed for. While the risk is low, you are proceeding at your own risk. Recovery information is available on Center Code (linked in the main README).

## What You'll Get

After following these guides, you'll have access to powerful tools including:
- **Sliced Kit Creation**: Turn audio files into drum kits
- **Chord Generation**: Create chord variations from samples  
- **Preset Editing**: Modify Drift synthesizer parameters and macros
- **MIDI Import**: Upload MIDI files to create new sets
- **Sample Processing**: Reverse audio, time-stretch, and more
- **Set Management**: Upload and restore Move Sets
